CYBER CASE FILE V2 - GAME AN TOÀN THÔNG TIN

Cách chạy:
1. Mở thư mục CyberCaseFile_V2
2. Bấm đúp file index.html
3. Chơi bằng Chrome hoặc Edge

Tính năng:
- Cốt truyện mở đầu
- Nhập tên điều tra viên
- 15 câu mỗi lượt
- Random từ ngân hàng 100 câu
- 20 giây/câu
- Bỏ qua/hết giờ sẽ không hiện lại trong lượt chơi
- Âm thanh đúng/sai/hết giờ/khởi động
- Combo, điểm thưởng theo thời gian
- Server HP, EXP, danh hiệu
- Kho bằng chứng
- Achievement cuối game
- Leaderboard lưu trên cùng máy/cùng trình duyệt

Lưu ý:
Leaderboard dùng LocalStorage nên chỉ lưu trên trình duyệt đang dùng.
